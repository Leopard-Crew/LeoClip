# LeoClip Release Checklist

LeoClip releases are verified through:

```sh
tools/release_check.sh
````

The script must be run on Mac OS X Leopard with Xcode 3.1.4 installed.

## Automated Checks

The release check performs:

- clean Release build
    
- clipboard history smoke test
    
- bundled `Localizable.strings` count check
    
- bundled `InfoPlist.strings` count check
    
- DMG creation
    
- DMG verification through `hdiutil verify`
    
- SHA-256 checksum generation
    

A successful run ends with:

```text
LeoClip release check passed.
```

## Manual UI Smoke Test

After the automated release check, perform a short manual UI test:

1. Start `build/Release/LeoClip.app`.
    
2. Copy `Alpha`.
    
3. Copy `Beta`.
    
4. Copy `Alpha` again.
    
5. Open the LeoClip status menu.
    
6. Verify that `Alpha` is on top and not duplicated.
    
7. Restore entries through Command-number shortcuts.
    
8. Pause Clipboard History.
    
9. Copy another text and verify it is not added.
    
10. Resume Clipboard History.
    
11. Clear History.
    
12. Quit LeoClip.
    

## Release Assets

A public release should include:

- `LeoClip-x.y.z-Leopard-PPC.dmg`
    
- `LeoClip-x.y.z-Leopard-PPC.dmg.sha256`
    

The DMG and checksum are generated by:

```sh
tools/make_dmg.sh
```

`tools/release_check.sh` calls this script automatically.

## Versioning

Use:

- patch release for packaging, localization, or release-process fixes
    
- minor release for internal architecture improvements or testability improvements
    
- major release only after the app is considered stable enough for general Leopard/PPC use
    

## Privacy Rule

LeoClip keeps clipboard history in memory only.

Release work must not add persistence, sync, network behavior, telemetry, or updater behavior unless explicitly documented and intentionally accepted.

## 0.7.0 Manual UX Checks

After running tools/release_check.sh, verify the status menu manually:

    About LeoClip is the first menu item.
    About LeoClip is separated from the rest of the menu.
    About opens the standard Cocoa About panel.
    The About panel appears in front of other application windows.
    The copyright text appears on two lines.
    Clear History has no ellipsis.
    Clear History asks for confirmation before deleting stored clips.
    Cancel leaves history untouched.
    Clear History removes stored clips only after confirmation.
    The current clipboard contents are not changed by clearing history.
    Quit LeoClip remains the last menu item.

This manual check exists because Leopard does not reliably expose third-party NSStatusItem menus through System Events UI scripting.
